# Botframework-webchat-embed Floating Window Sample
This is a native web chat floating widget built by Microsoft. It's the simplest way to add to a chat widget to your web site. The full documentation on how to use and customise this widget is found here: https://github.com/microsoft/botframework-webchat-embed

Use the docs to see the current limitations e.g. at the time of adding this, it only support anonymous authentication.
